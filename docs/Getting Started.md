Use /hunt **TWICE** if its your first time using the bot! (There's a bug registering new users)

You start off with one egg that you can hatch to begin hunting for hosts!

You'll need both hosts and eggs to get started. You should try to claim many eggs, as you can sell eggs to obtain Royal Jelly.

Use __/hunt__ to hunt for a potential host! Claim eggs when they spawn in the designated channel randomly, you might have to race others!

Different eggs will hatch different facehuggers which may require different hosts to evolve.
## Hatching and Evolving
You'll need to hatch eggs that you've claimed with the __/hatch__ command.
Once they are ready after a set period of time, use __/eggs list__ to view your current eggs, and claim the ones that are hatchable. 
After claiming, you'll then use the __/evolve__ command to have a facehugger implant a host, which is where the hosts you have hunted down come in. This command will also be used to evolve low evolutions into higher evolutions at the cost of Royal Jelly! (Base evolutions are free)
## Hives
The option to create a hive is available after you evolve atleast one xenomorph past the egg stage, you can create one with the __/hive create__ command. 
After that, you can view it with the __/hive stats__ commands. As you progress, you'll be able to upgrade modules and manage your hive with the various __/hive__ commands that are available.
## Information
You can use the command __/encyclopedia__ to view the various eggs, xenomorphs and hosts, as well as their rarities and the amount currently in the server.
The __/pathways__ command can be used to see the available evolution pathways and which eggs belong to each route. If a specific host is required, it will also be stated in this.
If you are curious about a specific egg in your inventory and information such as hatching time, sell value, etc., you can see these through the use of the __/eggs info__ command.
If you encounter a bug or have suggestions, you can use __/support-server__ to receive a link.
